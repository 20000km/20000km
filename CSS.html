코딩

CSS

  • ; 효과를 구분하기위해 마지막에 넣어줌
  • (style) a {color;black;}(/style) 이 페이지 전체 a태그를 선택해서 글자색상을 블랙으로 선언
  • (a style="black") (/a) a 태그만 글자색상을 블랙으로
  • text-decoration 택스트 장식 고르기
  • class 여러 텍스트를 클래스로 묶기 ex) class="saw" , .saw 로 쓰임
  • id 단 한 텍스트에 id값 ex) id="active" , #active
  • id(#) > class(.) > 그냥선택자 포괄적인 개념이 젤 약하다
  • 같은 선택자를쓰면 맨밑에쓴 선택자가 이긴다
  • border 테두리
  • width 두께
  • color 컬러
  • border-style 테두리 선의 종류
  • h1은 block element a는 inline element
  • h1, a{border:solid red 5px} h1이랑 a는 5px solid red로 테두리를 감싼다
  • padding 테두리와 글자사이 간격
  • margin 여백
  • width 폭 hi 테두리를 무시할수있음
  • div 무색무취 block element
  • span 무색무취 inline element
  • grid 레이아웃 최신기술
  • fr 비율 ex) grid-template-columns:1fr 1fr 화면을 2등분해서 1:1로 나누어짐
